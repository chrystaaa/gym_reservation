# Gym Reservation Bot

This is a Python script that automates gym reservation bookings for the University of Cyprus Sports Center.

## Features
- Automates gym reservations using Selenium
- Refreshes every 10 seconds to check for available slots
- Books a reservation automatically
- Runs in a loop until a reservation is successfully made

## Requirements
- Python 3 installed
- Google Chrome installed
- ChromeDriver installed
- Selenium module installed 

## Notes
- The bot will keep running until a reservation is successfully made.
- Make sure your Chrome profile is correctly configured.
- You can modify the `CHECK_INTERVAL` variable to adjust the refresh rate.
